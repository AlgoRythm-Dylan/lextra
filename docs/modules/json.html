JSON

The module used for parsing and serializing data in JSON format

Item name Item description
parse
(string, number)		 Given any string, either parse the valid JSON data into a Lua table, or fail and raise an error. Returns two items: [1] the Lua table, [2] the index of the last character of the JSON string where the parser stopped. It returns the second item mainly for parsing purposes (JSON.parse is recursive). Second (optional) parameter is where in the string to start parsing from.
stringify
(table, boolean)		 Given any Lua table, returns a valid JSON string. Second (optional) parameter expects a boolean and if true, enables an array check. Any table that CAN be represented as a JSON array WILL be. This is false by default
